Tools I've Built

Self-Generating Novel Idea Machine

I wanted to experiment with trying to get LLM's to output original thoughts, so I built this tool. It takes a natural language input consisting of a topic area, then it systematically generates ideas which it compares to a rubric aligned to threshold values. It cycles through ideas until it finds a minimum number of ideas that score as ''novel'' on the rubric, presenting them neatly to the user..

Book Chapter Automagic Formatter

I wanted to be able to paste in a chapter for a book (or multiple) in absolute plaintext - then have it output a fully styled document by using A.I. to intelligently process the context. Doing so enabled my tool to apply font styling including bold/italic/capitalization, as well as line spacing, paragraph breaks, block quotes, chapter headings, and citations! I tried to break it and it can handle anything you throw at it short of quantum languaging.

Scientific Research Study Proxy

Don't have actual humans to run a study on but need to test a hypothesis? That's why I built this. You build a config with JSON consisting of research area, methodology, personas of your subjects, how to score them, number of iterations, etc and it runs pretty dang helpful studies against digital humans and then analyzes the results.
